# Intelligent_Home
This project implements a smart door lock system based on the Arduino Uno.

It combines password authentication, environmental monitoring, and actuator control to provide a simple and secure embedded access system.

The system allows users to lock or unlock a door using a keypad, while also detecting gas leaks using an MQ-4 gas sensor and triggering an alarm when necessary.

A 16x2 LCD display is used to display system messages and user instructions, while a servo motor controls the physical locking and unlocking mechanism.
